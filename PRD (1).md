# PRD - Smart Scraper Platform

## Overview
AI-powered multi-agent scraping platform with orchestration, semantic understanding, and export capabilities.

## Core Features
- Smart scraping (Playwright + adaptive extraction)
- Multi-agent orchestration
- Vector-based semantic search
- Export (Excel, PDF, Word)
- Multi-user SaaS system

## Value
Automates data extraction and turns websites into structured, usable intelligence.
